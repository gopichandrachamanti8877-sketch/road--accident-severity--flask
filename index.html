Project Purpose

The purpose of this project is to demonstrate how deep learning can support road accident severity assessment using image data. Human review of CCTV footage can be slow and inconsistent, while an AI prototype can provide an initial automated severity estimate.

This system does not replace emergency decision-making. It demonstrates how machine learning can assist intelligent transportation systems by identifying whether an accident image appears to show minor, substantial, or critical impact.

Important Note

This is an academic prototype for the Machine Learning and Smart Systems project. Predictions depend on image quality and similarity to the training dataset.

Real-world severity assessment requires video sequence, vehicle speed, medical impact, road conditions, and expert human verification.
