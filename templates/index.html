📘 Project Overview

This project presents an AI-powered road accident severity classification system using Convolutional Neural Networks. The system classifies accident images into Minor Impact, Substantial Impact, and Critical Impact.

🔬 Research Problem

Manual accident severity assessment can be slow, inconsistent, and dependent on human judgement. This project investigates whether CNN-based image classification can support faster initial accident severity screening.

🎯 Research Objectives

  • Develop a CNN-based road accident severity classification model.
  • Compare Custom CNN, MobileNetV2, and EfficientNetB0.
  • Evaluate performance using accuracy, precision, recall, and F1-score.
  • Select the best-performing model for deployment.
  • Deploy the trained model as a Flask web application.
  • Provide prediction confidence and class probabilities.
